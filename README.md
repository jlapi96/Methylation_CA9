# README: In Silico Analysis of CA9 gene Methylation: Insights into its Role in Regulating CA IX Expression and its Clinical Associations

This repository contains the complete Python script for finding the methylation sites of CA9 gene and their location, together with merging publicaly available datasets used in this manuscript.  

All scripts are fully self‑contained and, after adjusting file paths for concete cancer type, can be run end‑to‑end to gain the CSV files which will contain data used in the manuscript. The scripts automatically check for and install missing packages. Code for statistical analysis can be provided upon request as it is not publicly available.
